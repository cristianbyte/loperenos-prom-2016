# Reencuentro Prom 2016

Proyecto en Astro con Tailwind y pequenas islas en Preact.

## Estructura

- `src/layouts/Layout.astro`: shell HTML global.
- `src/components/Header.astro`: encabezado estatico.
- `src/components/Gallery.astro`: composicion del grid.
- `src/islands/StudentCard.jsx`: tarjeta interactiva.
- `src/islands/LightboxIsland.jsx`: visor y navegacion.
- `src/data/students.js`: fuente de datos.

## Comandos

- `pnpm install`
- `pnpm dev`
- `pnpm build`
